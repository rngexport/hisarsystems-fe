# Vite + React + TypeScript + Tailwind

Vite, React, TypeScript ve Tailwind CSS ile hazır bir başlangıç şablonu.

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

Tailwind içerik dosyaları `tailwind.config.js` altında tanımlı. Global stiller `src/index.css`, örnek sayfa içeriği `src/App.tsx` dosyasında.
